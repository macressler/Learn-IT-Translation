Відсоток

Відсоток - одна сота частки. Позначається знаком «%». Використовується для позначення частки чого-небудь по відношенню до цілого. Наприклад, 17 % від 500кг означає 17 частин по 5кг кожна, тобто 85кг. Справедливо також твердження, що 200 % від 500кг є 1000кг, оскільки по відношенню до половини тонни, тонна відповідає 2×100 %.

Співвідношення відсотків і десяткових дробів:

  • 0 % = 0
  • 0,07 % = 0,0007
  • 45,1 % = 0,451
  • 100 % = 1
